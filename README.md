

# Automated Quality Inspection System

This project is an industrial computer vision application developed using Python, OpenCV, and Tkinter. The system performs real-time quality inspection of mechanical parts through a webcam feed and automatically identifies structural defects using contour analysis, convex hulls, and convexity defect detection.

## Features

* Real-time webcam inspection
* Industrial GUI dashboard
* Automated PASS/FAIL classification
* Convex Hull and Convexity Defect Analysis
* PLC Signal Simulation
* Defect Bounding Boxes
* Screenshot Capture
* Adjustable Inspection Parameters
* Triple Verification Logic for reliable defect detection
* Live Performance Metrics

## Technologies Used

* Python
* OpenCV
* NumPy
* Tkinter
* Pillow (PIL)

## Project Workflow

Input → Webcam Feed

Processing →

* Grayscale Conversion
* Gaussian Blur
* Thresholding
* Contour Detection
* Convex Hull Analysis
* Convexity Defect Detection

Output →

* PASS / FAIL Decision
* Defect Highlighting
* PLC Signal Generation
* Inspection Statistics Dashboard
